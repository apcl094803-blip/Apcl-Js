1. HTML Element คืออะไร?

HTML Element คือหน่วยพื้นฐานของหน้าเว็บ ประกอบด้วย Tag เปิด เนื้อหา และ Tag ปิด

<a>…</a>
คุณสมบัติรายละเอียด
สร้างโดยTim Berners-Lee (1991)
มาตรฐานW3C / WHATWG (HTML5 → Living Standard)
ไฟล์.html หรือ .htm
2. HTML Document Structure
<!DOCTYPE html>
<html lang="th">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>ชื่อหน้าเว็บ</title>

</head>
<body>
  <!-- เนื้อหาอยู่ที่นี่ -->
</body>
</html>
Tagคำอธิบาย
<!DOCTYPE html>บอก browser ว่าเป็น HTML5
<html>Root element ของทุกหน้า
<head>ข้อมูลของหน้าเว็บ (ไม่แสดงผล)
<meta charset>กำหนดการเข้ารหัสตัวอักษร (รองรับภาษาไทย)
<meta viewport>ทำให้รองรับมือถือ
<title>ชื่อที่แสดงบน tab browser
<body>เนื้อหาที่แสดงผลบนหน้าเว็บ
3. HTML Tag
ประเภท Tagตัวอย่างคำอธิบาย
Opening Tag<a>เริ่มต้น element
Closing Tag</a>จบ element (มี / นำหน้า)
Contentข้อความหรือ HTMLเนื้อหาอยู่ระหว่าง tag
Self‑Closing Tag<img> <br> <input>ไม่มี tag ปิด
4. HTML Attributes

Attribute คือข้อมูลเพิ่มเติมที่ใส่ใน Opening Tag เพื่อกำหนดคุณสมบัติของ element

<a href="http://google.com/" target="_blank" title="Google">คลิกที่นี่</a>
Attributeใช้กับคำอธิบาย
href<a>ลิงก์ปลายทาง
src<img>, <script>ที่อยู่ไฟล์
alt<img>ข้อความแทนรูปภาพ
classทุก elementกำหนดกลุ่ม CSS
idทุก elementชื่อเฉพาะของ element
styleทุก elementกำหนด CSS ตรงๆ
target<a>_blank เปิดแท็บใหม่
type<input>ประเภทของ input
placeholder<input>ข้อความตัวอย่างใน input
disabled<input>, <button>ปิดใช้งาน element
required<input>บังคับกรอก
action<form>URL ที่รับข้อมูล form
method<form>GET หรือ POST

13. <div> และ <span>

  <p>ข้อความ <span style="color:red">สีแดง</span> ปกติ</p>

14. Inline vs Block Elements
📦 Block Elements
  • เริ่มต้นบนบรรทัดใหม่เสมอ
  • กินพื้นที่เต็มความกว้างของ container
  • ตัวอย่าง: <div>, <p>, <h1>-<h6>, <ul>, <li>, <section>, <header>, <footer>, <article>, <nav>, <aside>, <table>, <form>, <hr>
📝 Inline Elements
  • ไม่ขึ้นบรรทัดใหม่ — เรียงต่อกันในแนวนอน
  • กินพื้นที่เท่าเนื้อหาข้างใน
  • ตัวอย่าง: <span>, <a>, <strong>, <em>, <img>, <button>, <input>, <label>, <br>, <code>, <q>, <cite>
🔄 Inline-block

display: inline-block ทำให้ element อยู่แนวเดียวกัน แต่กำหนด width/height ได้

<div>Div 1 (block)</div><div>Div 2 (block)</div>
<span>Span A (inline)</span><span>Span B (inline)</span>
<span style="display:inline-block; width:100px;">Inline-Block</span>
✨ การเปลี่ยนพฤติกรรมด้วย CSS: display: block หรือ display: inline

21. Emmet for HTML — เขียนไวระดับเทพ

Emmet เป็นปลั๊กอินใน VS Code, Sublime Text, Atom ที่ช่วยให้เขียน HTML/CSS ได้เร็ว โดยใช้ CSS selector syntax แล้วกด Tab หรือ Enter เพื่อขยายเป็นโค้ดจริง

🔥 Syntax พื้นฐาน
Emmetผลลัพธ์ (กด Tab)
! หรือ html:5โครงสร้าง HTML5 ครบ
div<div></div>
a<a href=""></a>
img<img src="" alt="">
ul>li*5<ul><li></li>…5 ตัว</ul>
div.wrapper<div class="wrapper"></div>
p#main<p id="main"></p>
a[href=https://google.com]<a href="https://google.com"></a>
loremข้อความตัวอย่าง Lorem ipsum
lorem10Lorem 10 คำ
ul>li.item$*3class="item1", "item2", "item3"
a{click here}<a href="">click here</a>
📦 ตัวดำเนินการ (Operators)
  • # → id, . → class
  • > (ลูก), + (พี่น้อง), ^ (ขึ้นไปหนึ่งระดับ)
  • * (คูณ), () (กลุ่ม)
  • $ (เลขเรียง), @- (เรียงจากมากไปน้อย)
⚡ ตัวอย่าง Emmet ที่ใช้บ่อย
<!-- Emmet: header>nav>ul>li*3>a -->
<header><nav><ul><li><a href=""></a></li><li><a href=""></a></li><li><a href=""></a></li></ul></nav></header>

<!-- Emmet: form.login-form>input:text+input:email+button[type=submit]{Send} -->
<form action="" class="login-form">
  <input type="text" name="" id="">
  <input type="email" name="" id="">
  <button type="submit">Send</button>
</form>

<!-- ใช้ lorem สร้างข้อความตัวอย่าง -->

22. HTML5 Features (API & Elements)

HTML5 นำเสนอ API และ element ใหม่ๆ ที่ช่วยให้เว็บแอปพลิเคชันมีประสิทธิภาพมากขึ้น โดยไม่ต้องพึ่งปลั๊กอิน (Flash, Silverlight)

📦 Web Storage (localStorage & sessionStorage)

เก็บข้อมูลฝั่ง client ได้มากกว่า cookie (5-10MB) และไม่ต้องส่งไปกับ request ทุกครั้ง

// เก็บข้อมูล
localStorage.setItem('key', 'value');
sessionStorage.setItem('temp', 'data');

// อ่านข้อมูล
let val = localStorage.getItem('key');

// ลบ
localStorage.removeItem('key');
localStorage.clear();
🎨 Canvas & SVG

Canvas ใช้ JavaScript วาดกราฟิก 2D แบบ pixel เหมาะกับเกม, กราฟ, ภาพเคลื่อนไหว
SVG (Scalable Vector Graphics) ใช้ XML วาดรูป vector ขยาย size ไม่แตก เหมาะกับไอคอน, แผนที่

<canvas id="myCanvas" width="200" height="100"></canvas>
<svg width="100" height="100"><circle cx="50" cy="50" r="40" fill="red"/></svg>
📍 Geolocation

เข้าถึงตำแหน่งทางภูมิศาสตร์ของผู้ใช้ (ต้องได้รับอนุญาต)

navigator.geolocation.getCurrentPosition((pos) => {
  console.log(pos.coords.latitude, pos.coords.longitude);
});
🖱️ Drag-and-Drop API

ทำให้ element สามารถลากและวางได้

<div draggable="true" ondragstart="drag(event)">ลากฉันได้</div>
<div ondrop="drop(event)" ondragover="allowDrop(event)"></div>
⚙️ Web Workers

ทำงาน background โดยไม่กระทบ UI (เหมาะกับงานหนัก)

// main.js
const worker = new Worker('worker.js');
worker.postMessage('start');
worker.onmessage = (e) => console.log(e.data);
🔄 History API (pushState, replaceState)

เปลี่ยน URL แบบไม่รีเฟรชหน้า เหมาะกับ Single Page Application

history.pushState({page: 2}, "title 2", "/page2");
📡 Server-Sent Events (SSE)

รับข้อมูลจาก server แบบ real-time (server push) เหมาะกับ notification, stock update

const evtSource = new EventSource("/stream");
evtSource.onmessage = (e) => console.log(e.data);
🎬 Audio / Video API ขั้นสูง

ควบคุมการเล่นแบบ JavaScript เช่น .play(), .pause(), .currentTime, volume

🧩 Web Components (Custom Elements, Shadow DOM)

สร้าง element ใหม่ reusable ได้เอง

📱 Responsive Images (srcset, picture element)
<img srcset="img-small.jpg 480w, img-large.jpg 1080w" sizes="100vw">
